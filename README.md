billable
========

Tracks billable hours.


### Installation

```sh
$ npm install -g billable
```

### Start Clock

```bash
$ billable start
```


### Stop Clock

```bash
$ billable stop
```


### Get Log

```sh
$ less ./billable/anonymous_timesheet
```
